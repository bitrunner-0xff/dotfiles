# My Config Files

## Content

- `Hypr` - Window manager
- `Kitty` - Terminal
- `Mako` - Notification daemon
- `Nvim` - Terminal text editor
- `Ranger` - Terminal file manager
- `rose-pine` - Refind theme
- `Satty`
- `Waybar` - Top bar
- `Wofi` - Program runner

## How to install

```sh
git clone https://github.com/bitrunner-0xff/dotfiles.git
```
